# Lugano 2025 - Breez SDK Nodeless Workshop

This repository contains materials for the "Breez SDK Nodeless - Plan B Edition" workshop, focusing on integrating Bitcoin Lightning Network payments into web applications without running your own node.

## Workshop Overview

The workshop demonstrates practical implementation of nodeless Bitcoin payments using the Breez SDK Liquid. Participants learn how to:

- Set up a Breez SDK wallet in a web application
- Handle Bitcoin transactions (receive/send sats)
- Integrate payment functionality into user interfaces
- Manage wallet states and transaction history

## Demo Application

The demo is a modernized Space Invaders game built with React and TypeScript, featuring:

### Core Game Features
- Classic Space Invaders gameplay with player spaceship and invader waves
- Real-time collision detection and projectile mechanics
- Particle effects and audio feedback
- Responsive game grid and scoring system

### Breez SDK Integration
- **Wallet Management**: Connect to Breez SDK for nodeless Bitcoin operations
- **Top-Up Modal**: Add sats to your wallet balance
- **Transaction History**: View past payments and receipts
- **Profile Management**: Handle user wallet profiles
- **Real-time Balance**: Display current sats balance during gameplay

### Technical Stack
- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: CSS modules
- **Audio**: Web Audio API for game sounds
- **Bitcoin Integration**: Breez SDK Liquid (WebAssembly)
